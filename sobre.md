# Extrator Mestre

O Extrator Mestre é um sistema desenvolvido como uma API FastAPI focada na extração de dados de diversos formatos de arquivos. 

## Funcionalidades e Suporte
- Extração de dados de **PDFs**
- Extração de dados de **Imagens**
- Extração de dados de arquivos do **Office (DOC, DOCX e XLSX)**

## Infraestrutura
O sistema possui suporte nativo ao **Docker**, permitindo uma implantação fácil, isolada e escalável.
